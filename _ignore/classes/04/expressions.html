Now That We've Seen Some Code…

Let's revisit some terminology.

Statements

A statement is:

  • an instruction that the interpreter can execute
  • think of a line of code
  • some examples of statements that we've seen are:
    • assignment statement
    • if statement
  • statements can contain expressions

Expressions

An expression is:

  • a combination of values, variables, operators, function calls
  • when evaluated, produces a result (a value)
  • some examples of expressions:
    • 5 + 5
    • int("24")
    • input(">")
