Additional Turtle Methods

  • clear() - removes drawing from screen
  • hideturtle() - does not show turtle (triangle)
  • circle(int) - draws a circle using supplied radius

Additional Screen Methods

  • tracer(int) - when called with 0, turns off animation of turtle movement
  • update() - when tracer is 0, use this to refresh the screen (show drawing)
  • ontimer(function, int) - automatically call a function after some interval of time
