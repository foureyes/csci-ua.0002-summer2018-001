Detour - Some Definitions Revisited

The previous slides mentioned function call, parameters and arguments. What does it mean to call a function? What is a parameter, and what is an argument? →

  • calling a function - running or executing the function
  • argument - a piece of data that is passed into a function
  • a parameter - is a variable in a function that receives an argument that's passed
    • within a function the values passed in can be referred to by parameter name

….And More Definitions

What is a function definition, header, and body? What is meant by saying a block of code? →

  • function definition - all of the code for a function
  • function header - the beginning of the function: its name and optional list of parameters
  • function body - the code that gets executed when a function is called..
  • block - a set of statements that belong together as a group; indentation marks the beginning and end of a block

Variables Inside a Function

What's a local variable? What's scope? →

  • local variable - variable created inside a function's definition
    • only accessible within that function
    • cannot be changed from outside of the function
    • does not overwrite variables of the same name outside of the function (or in other functions)
  • scope - the part of the program where a variable can be accessed
