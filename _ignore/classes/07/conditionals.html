Truthiness

See this crazy chart on the intrinsic boolean value of various types. The following values are considered false:

  • None
  • False
  • 0 of any numeric type (0.0, 0)
  • empty mapping or sequence type (We'll look at these later) - this includes the empty string '', "", etc.
  • some instances of user defined types based on bool method
