Random - Choice, Shuffle

  • What function from the random module can be used to retrieve a random element from a list? →
  • What function from the random module can be used to mix up a list? →
  • choice(my_list): retrieves a random item from the supplied list
  • shuffle(my_list): randomly changes the order of the elements in the list passed in (in place)

Generating Random Elements from a List

Using the alphabetically sorted list of sounds below…

  1. Print 20 sounds randomly →
  2. Print each sound exactly once in random order →

Generating Random Elements from a List Solution

import random

sounds = ['beep', 'buzz', 'fizz', 'honk'] for i in range(20): print(random.choice(sounds))

print('—–')

random.shuffle(sounds) for sound in sounds: print(sound)
